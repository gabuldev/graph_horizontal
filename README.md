# graph_horizontal

  

The graph horizontal with tooltips and three types.

  

GraphType.one

GraphType.two

GraphtType.three

  

## Example

  

```dart

GraphHorizontal(

type:  GraphType.three,

items: [

GraphItem(value:  50, label:  "A"),

GraphItem(value:  5, label:  "B"),

],

),

```
